# Vue Gantt Chart Library



## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build library
npm run build

# Type check
npm run type-check

# Lint code with oxlint
npm run lint

# Format code with oxfmt
npm run format

# Check formatting without modifying files
npm run format:check

# Run Storybook
npm run storybook

# Build Storybook
npm run build-storybook

# Run tests
npm run test
```
